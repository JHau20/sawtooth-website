System Administrator’s Guide¶

This guide explains how to install, configure, and run Hyperledger Sawtooth on a Ubuntu system for proof-of-concept or production use in a Sawtooth network.

  • See Setting Up a Sawtooth Network to configure and run a Sawtooth node with either PBFT or PoET simulator consensus.

  • See Using Sawtooth with PoET-SGX to configure and run a Sawtooth node with PoET consensus on a system with Intel® Software Guard Extensions (SGX).

This guide also includes optional administration procedures, such as how to restrict transaction types; configure user, client, and validator permissions; and display Sawtooth metrics with Grafana.

Other sections in this guide summarize dynamic consensus settings and explain how to use Sawtooth configuration files.
