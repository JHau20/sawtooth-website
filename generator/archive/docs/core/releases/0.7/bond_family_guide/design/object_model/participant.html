object-type: participant¶

Attributes¶

object-id
See Common Attributes.
object-type
The string ‘participant’.
creator-id
See Common Attributes.
key-id

The public identifier derived from the key used by this participant to sign messages. In other systems such as bitcoin, where this might be used as a destination, the key-id is also called the ‘address’ or ‘wallet address’.

Indexed, Unique

username

The username of the participant.

Indexed, Unique

firm-id

The object-id of the participant’s organization.

Optional

Example JSON¶

{
    "object-id": "<OBJECT_ID>",
    "object-type" : "participant",
    "creator-id": "<OBJECT_ID>",
    "key-id": "<KEY_ID>",
    "username" : "tjsmith",
    "firm-id" : "<OBJECT_ID>"
}
