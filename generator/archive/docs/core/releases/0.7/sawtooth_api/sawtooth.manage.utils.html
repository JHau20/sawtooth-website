sawtooth.manage.utils.get_executable_script(script_name)[source]¶

Searches PATH environmental variable to find the information needed to execute a script. :param script_name: the name of the ‘executable’ script

Returns:A list containing the python executable, and the full path to the script. Includes sys.executable, because certain operating systems cannot execute scripts directly.
