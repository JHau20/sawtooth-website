HTTP Routing Table

/batch_statuses | /batches | /blocks | /peers | /receipts | /state | /status | /transactions
 
/batch_statuses
GET /batch_statuses Fetches the committed statuses for a set of batches
POST /batch_statuses Fetches the committed statuses for a set of batches
 
/batches
GET /batches Fetches a list of batches
GET /batches/{batch_id} Fetches a particular batch
POST /batches Sends a BatchList to the validator
 
/blocks
GET /blocks Fetches a list of blocks
GET /blocks/{block_id} Fetches a particular block
 
/peers
GET /peers Fetches the endpoints of the authorized peers of the validator
 
/receipts
GET /receipts Fetches the receipts for a set of transactions
POST /receipts Fetches the receipts for a set of transactions
 
/state
GET /state Fetches the data for the current state
GET /state/{address} Fetches a particular leaf from the current state
 
/status
GET /status Fetches information pertaining to the status of the validator
 
/transactions
GET /transactions Fetches a list of transactions
GET /transactions/{transaction_id} Fetches a particular transaction
